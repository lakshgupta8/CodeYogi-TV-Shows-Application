# TV SHOWS Search App

A show discovery applicbation uilt with a **Gray-Black-Brown** aesthetic.

## Features

- **Real-time Discovery**: Search through a vast library of TV shows using the TVMaze API.
- **Deep Insights**: Detailed show information, including genres, summaries, and ratings.
- **Cinematic Experience**: Premium dark-mode design with amber accents and smooth transitions.
- **Advanced State Management**: Powered by Redux and Redux Saga for robust, asynchronous data handling.
- **Responsive Design**: Fully optimized for various screen sizes using Tailwind CSS.

## Tech Stack

- **Frontend**: React (with Vite)
- **State Management**: Redux, Redux Saga
- **Styling**: Tailwind CSS
- **Routing**: React Router v6
- **Data Fetching**: Axios
- **API**: TVMaze API

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/lakshgupta8/CodeYogi-TV-Shows-Application.git
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   bun install
   ```

3. Start the development server:

   ```bash
   npm run dev
   # or
   bun dev
   ```
